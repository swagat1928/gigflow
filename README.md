GigFlow is a mini freelance marketplace platform where clients can post gigs, freelancers can bid on them, and clients can hire freelancers with real-time notifications using Socket.IO.

Features:
Authentication:
User registration & login

Gigs:
Create gigs (clients)
View all open gigs
Search gigs by title
View gig details

Bidding System:
Freelancers can submit bids
Clients can view bids on their gigs
Clients can hire a freelancer
Bid & gig status updates

Real-Time Notifications:
Freelancer receives instant notification when hired


Tech Stack:

Frontend
React + Vite
Redux Toolkit
Tailwind CSS
Axios
Socket.IO Client

Backend:
Node.js
Express.js
MongoDB 
JWT Authentication
Socket.IO


Installation & Setup:

1. Clone the repository

git clone https://github.com/your-username/gigflow.git
cd gigflow

2. Backend Setup

cd backend
npm install

Create a .env file using .env.example:

npm run dev

3. Frontend Setup

cd frontend
npm install
npm run dev
